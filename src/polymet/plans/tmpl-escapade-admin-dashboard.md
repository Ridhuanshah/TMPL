# TMPL Escapade Admin Dashboard

## User Request
Create a comprehensive, high-fidelity admin CMS dashboard for TMPL Escapade, a Malaysian travel agency. The dashboard should manage travel packages, bookings, users, stores, coupons, reviews, and provide analytics across all operations.
